# UI/KIT

Планирую создать свой ui-kit чтобы быстрее разворачивать разработку новых приложений.
На это есть несколько причин:
- Я использую собственную палитру с своими названиями для переменных цветов.
- Есть необходимость создания уникальности моим приложениям.
