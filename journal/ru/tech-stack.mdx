Мне показалось что на главной странице не хватает секции с технологиями, которые я использую в разработке, поэтому я принялся за его разработку.
При поиске дизайна для нового блока я вспомнил про один дизайн, который я уже видел на [домашней странице Vercel](https://vercel.com/home), поэтому я взял его за основу и адаптировал под свой сайт.

При наведении на любой блок, кроме двойного, иконка подсвечивается и триггерит вывод названия в двойной блок.

<TechStack title="Двойной блок" />

Компонент принимает массив, который содержит информацию о стэке, для этого я создал тип Stack.

```
type Stack = {
  label: string
  value: string
  icon: ReactNode
  color?: string
  y: number
}
```
Была релизованна поочередная подсветка стэка, для этого был разработан компонент `<AutoSwitcher />`

```
<AutoSwitcher
  stop={stop} // Для остановки переключения
  index={index} // Текущих индекс
  maxIndex={maxIndex} // Последний индекс, чтобы не выйти за пределы
  onIndex={onIndex} // Для обновления текущего индекса
/>
```


```
<Wrapper className="w-full grid lg:grid-cols-4 grid-cols-2 grid-rows-3 lg:grid-rows-2"
  onEnter={() => {
    setIsOut(false)
  }}
  onLeave={() => {
    setSelected(null)
    setIsOut(true)
  }}
>
  <AutoSwitcher stop={!isOut} index={index} maxIndex={maxIndex} onIndex={setIndex} />
  <Header title={title}>
    <HeaderList y={y} selected={selected} stack={stack} />
  </Header>
  {
    stack
      .map(
        (item, techIndex) => {
          const isSelected = techIndex === index
          return (
            <Item
              key={"tech#" + techIndex}
              stack={item}
              onChange={setSelected}
              className={isAutoMode ? isSelected ? icon["active-icon"] : "" : icon["icon-wrapper"]}
            />
          )
        }
      )
  }
</Wrapper>
```

Одно из самых интересных было создание иконок, их внешним видом я вднохновился с [домашней странице Vercel](https://vercel.com/home).
