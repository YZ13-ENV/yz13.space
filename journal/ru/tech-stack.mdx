# Обновление

В недавнем обновлении появилась новая сецкция "Стэк", которая состоит из компонента `<TeckStack />`.

<Image
  className="rounded-xl overflow-hidden border"
  src="/journal/tech-stack/grid.png"
  width={600}
  height={400}
  alt="tech-stack-grid"
/>

Сам компонент является сеткой, где первый блок занимает две ячейки по горизонтали, остальные по одной.
При наведении на любой блок, кроме двойного иконка подсвечивается и триггерит вывод названия в двойной блок.
